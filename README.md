# Волк ловит яйца — Telegram Mini App / HTML5

Готовый статический проект для размещения на GitHub Pages и подключения как Telegram Mini App.

## Что внутри

- `index.html` — основная HTML-страница.
- `styles.css` — адаптивный дизайн под мобильный экран Telegram.
- `game.js` — логика игры, управление касанием куриц, счёт, жизни, уровни и звук.
- `assets/wolf-pos-1.png` … `assets/wolf-pos-4.png` — четыре изображения, где волк стоит у разных лотков.
- `.nojekyll` — файл для корректной публикации статического сайта на GitHub Pages.

## Управление

Кнопок навигации нет. Волк перемещается при нажатии прямо на куриц:

- курица 1 — верхний левый лоток;
- курица 2 — верхний правый лоток;
- курица 3 — нижний левый лоток;
- курица 4 — нижний правый лоток.

## Звук

Звук сделан без внешних аудиофайлов через Web Audio API: короткие оригинальные ретро-сигналы в стиле карманных электронных игр.

На странице есть:
- включение / отключение звука;
- регулировка громкости.

## Размещение на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него все файлы из архива.
3. Откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите:
   - Source: **Deploy from a branch**;
   - Branch: **main**;
   - Folder: **/ root**.
5. Сохраните настройки.
6. Через 1–3 минуты GitHub Pages выдаст HTTPS-ссылку на игру.

## Подключение в Telegram

1. Откройте BotFather.
2. Создайте или выберите бота.
3. Добавьте Mini App / Web App и укажите HTTPS-ссылку GitHub Pages.
4. После публикации игра будет открываться внутри Telegram.

## Настройка сложности

В `game.js` можно изменить параметры:

```js
const GAME = {
  livesMax: 3,
  baseTravelTime: 3600,
  minTravelTime: 1750,
  baseSpawnEvery: 1450,
  minSpawnEvery: 720,
};
```

- `baseTravelTime` — скорость движения яиц на первом уровне.
- `minTravelTime` — максимальная скорость на старших уровнях.
- `baseSpawnEvery` — начальная частота появления яиц.
- `minSpawnEvery` — минимальный интервал между яйцами.

## Проверка зон нажатия

Для отладки зон куриц откройте страницу с параметром:

```text
?debug
```

Например:

```text
https://ваш-логин.github.io/ваш-репозиторий/?debug
```

На экране появятся прозрачные зоны, по которым срабатывает перемещение волка.

## Оптимизация

Картинки уменьшены до 960×720 и сохранены в WebP, поэтому игра быстрее загружается в Telegram Mini App и на GitHub Pages.
Текст подсказок усилен по контрасту для темных тем Telegram.
