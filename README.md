# Волк ловит яйца — HTML5 / Telegram Mini App

Готовая статическая HTML5-игра для размещения на GitHub Pages и подключения к Telegram Mini App.

## Что внутри

```text
nu_pogodi_telegram_game/
├── index.html
├── assets/
│   └── background.png
├── .nojekyll
├── .gitignore
└── README.md
```

## Возможности

- Игра на HTML5 Canvas без сборки и без серверной части.
- Фон и визуальный стиль основаны на предоставленной картинке.
- Ретро-звуки в стиле карманной игры «Электроника»: звук ловли, промаха, движения, старта, нового уровня и окончания игры.
- Звук синтезируется через Web Audio API, поэтому отдельные аудиофайлы не нужны.
- Кнопка включения/выключения звука.
- Ползунок регулировки громкости.
- Сохранение рекорда, громкости и режима звука в localStorage.
- Адаптация для Telegram WebView: `Telegram.WebApp.ready()`, `expand()`, тема, haptic feedback, отправка результата через `sendData()` при доступности.
- Управление с телефона и клавиатуры.

## Быстрый запуск локально

Откройте файл `index.html` в браузере.

Для проверки через локальный веб-сервер:

```bash
python3 -m http.server 8080
```

После запуска откройте:

```text
http://localhost:8080
```

## Размещение на GitHub Pages

1. Создайте новый репозиторий на GitHub, например `nu-pogodi-game`.
2. Загрузите в репозиторий все файлы из этой папки, включая `index.html`, `.nojekyll` и папку `assets`.
3. Откройте `Settings` → `Pages`.
4. В разделе `Build and deployment` выберите:
   - Source: `Deploy from a branch`;
   - Branch: `main`;
   - Folder: `/root`.
5. Сохраните настройки и дождитесь публикации.
6. Адрес игры будет похож на:

```text
https://USERNAME.github.io/nu-pogodi-game/
```

## Подключение к Telegram Mini App

1. Откройте `@BotFather`.
2. Создайте или выберите бота.
3. Через меню BotFather настройте Mini App / Web App и укажите HTTPS-ссылку GitHub Pages.
4. Для кнопки запуска в боте используйте URL опубликованной страницы.
5. При окончании игры кнопка «Отправить результат» вызывает `Telegram.WebApp.sendData()` и передаёт JSON:

```json
{
  "type": "nu_pogodi_score",
  "score": 25,
  "best": 40,
  "lives": 0,
  "ts": 1783520000000
}
```

На стороне бота нужно обработать входящие данные Mini App и сохранить/показать результат игрока.

## Управление

### На телефоне

Используйте четыре кнопки снизу:

- `1` — верх слева;
- `2` — верх справа;
- `3` — низ слева;
- `4` — низ справа.

### На клавиатуре

- `1` / `Q` — верх слева;
- `2` / `E` — верх справа;
- `3` / `A` — низ слева;
- `4` / `D` — низ справа;
- стрелки — выбор стороны и уровня;
- `M` — включить/выключить звук.

## Важное по звуку

На iOS, Android и в Telegram звук начинает работать только после первого действия пользователя: нажатия «Начать игру», кнопки управления или кнопки звука. Это нормальное ограничение мобильных браузеров.

## Настройка сложности

В `index.html` можно изменить параметры:

```js
state.spawnDelay = 1450;
```

И скорость яйца в функции `spawnEgg()`:

```js
speed: 0.00034 + Math.min(level, 10) * 0.000025 + Math.random() * 0.000055
```

Чем меньше `spawnDelay` и больше `speed`, тем сложнее игра.
